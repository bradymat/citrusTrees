Steps to get going

  1. `git clone https://github.com/bradymat/citrusTrees.git`
  
  2. `cd citrusTrees`
  
  3. `npm install`
  
  4. `knex migrate:latest`
  
  5. `knex seed:run`
  
  6. `npm start`
  
  7. type `localhost:3000` into your browser
  
then you can play around with adding citrus trees
